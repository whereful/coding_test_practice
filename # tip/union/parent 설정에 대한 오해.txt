1. 부모 배열을 제대로 설정하지 않으면 올바른 부모를 얻을 수 없다 => X

간선을 1번만 순회하는 경우 parent 배열이 제대로 설정되지 않을 수 있다

예를 들어 [0, 1, 1, 1, 1, 1, 5, 7]로 부모가 설정되었고
5의 부모는 1인 경우

제대로 설정된 결과는 [0, 1, 1, 1, 1, 1, 1, 7]이다

하지만 부모가 제대로 설정되지 않아도 find_parent로 조회하면 올바른 부모를 얻을 수 있다


2. 부모 배열을 제대로 설정하지 않으면 사이클을 판별할 수 없다 => X

1과 마찬가지로 find_parent로 조회하면 문제 없다
